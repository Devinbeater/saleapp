┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 1: PUSH TO GITHUB (2 minutes)                                          │
└──────────────────────────────────────────────────────────────────────────────┘

Open your terminal and run:

    git init
    git add .
    git commit -m "Ready for Render deployment"
    
    # Go to GitHub.com and create a new repository
    # Then run (replace YOUR_USERNAME with your GitHub username):
    
    git remote add origin https://github.com/YOUR_USERNAME/daily-register.git
    git branch -M main
    git push -u origin main

✅ Done? Move to Step 2


┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 2: DEPLOY ON RENDER (3 minutes)                                        │
└──────────────────────────────────────────────────────────────────────────────┘

1. Go to: https://render.com
2. Sign in with GitHub
3. Click "New +" button → Select "Blueprint"
4. Choose your repository: daily-register
5. Click "Apply"
6. Wait 3-5 minutes for deployment ☕

✅ Done? Move to Step 3


┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 3: INITIALIZE DATABASE (2 minutes)                                     │
└──────────────────────────────────────────────────────────────────────────────┘

1. In Render dashboard, click on your database: "daily-register-db"
2. Scroll to "Connections" section
3. Copy the "External Database URL" (starts with postgres://)
4. Open your terminal and run:

    psql YOUR_DATABASE_URL_HERE
    
5. Copy and paste the contents of "database-schema.sql" file
6. Press Enter to execute
7. Type \q and press Enter to exit

✅ Done? Your app is live!

→ Check DATABASE_URL is set in Render

Problem: App not loading
→ Check service logs in Render dashboard

Problem: Free tier sleeping
