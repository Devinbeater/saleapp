┌──────────────────────────────────────────────────────────────────────────────┐
│ HOW TO USE                                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

STEP 1: Enter Opening Cash
   → Look at sidebar → "Opening Cash" field
   → Enter starting cash amount (e.g., 5000)

STEP 2: Enter POS Transactions
   → Click any POS cell
   → Type amount (e.g., 500)
   → 🎉 MODAL APPEARS automatically!
   → Select entry type:
      • Sale 🛒 (regular transaction)
      • Return ↩️ (refund)
      • Salesman 👔 (collection)
      • Party 🤝 (payment)
   → Click "Confirm"
   → Badge appears on cell!

STEP 3: Enter Other Payments
   → Kotak QR column: Enter QR payments
   → Kotak Swipe column: Enter card payments
   → Debtors column: Enter credit sales
   → BharatPE field (sidebar): Enter BharatPE amount

STEP 4: Count Denominations
   → At end of day, count cash
   → Enter pieces for each denomination
   → Total calculates automatically

STEP 5: Check Reconciliation
   → Look at "Difference" field
   → Should show ₹0.00 ✅
   → If not zero, recount!


┌──────────────────────────────────────────────────────────────────────────────┐
│ CALCULATIONS EXPLAINED                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

Total Sale = Sum of all POS entries
   Example: 500 + 300 + 200 = ₹1,000

Debit Cash = Opening Cash + Total Sale
   Example: 5,000 + 1,000 = ₹6,000

Net Amount = Debit Cash - (KQR + KSW + Debtors + BharatPE)
   Example: 6,000 - (200 + 300 + 100 + 50) = ₹5,350

Collection = Denominations + KQR + KSW + Debtors + BharatPE
   Example: 5,350 + 200 + 300 + 100 + 50 = ₹6,000

Difference = Debit Cash - Collection
   Example: 6,000 - 6,000 = ₹0.00 ✅ PERFECT!

┌──────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

Problem: Modal doesn't appear
Solution: Make sure you're typing in POS column (first column)

Problem: Difference is not zero
Solution: 
   • Recount cash denominations
   • Check all digital payment entries
   • Verify Opening Cash is correct
   • Check for missed POS entries

Problem: Calculations not updating
Solution: 
   • Make sure you entered values correctly
   • Check browser console for errors
   • Refresh page and try again
